To use this application - there are two service calls..
  1. Get book that uses below url pattern 
    <project url>/rest/book/{id}
    where id is book id.

  2. Add book - send POST request to below URL 
    <project url>/rest/book/add
    with request body as below xml 
                    <book>
                    <id>2</id>
                    <author>Ravi Dasari</author>
                    <title>I didnt write this book ;)</title>
                </book>
